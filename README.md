# Electron Reminder

Windows desktop-приложение с напоминаниями.

Функции:
- задаете текст и время (в минутах);
- по истечении времени появляется полноэкранное окно поверх всех окон;
- окно закрывается только после нажатия `OK`.

## Запуск в режиме разработки

```bash
npm install
npm run start
```

## Сборка Windows установщика (.exe)

```bash
npm install
npm run dist:win
```

Готовый установщик будет в папке `release`.
